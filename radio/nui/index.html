DRP Radio

DRP Radio is our Motorola Radio emulator, using a custom made Mumble plugin.

It gives us a lot more possibilities, which currently allow such things as:

  • Digitial Trunked Radio System
  • Radio clicks
  • Talk enforcement
  • Emergency buttons
  • 911 calls

DRP Radio must be installed and running to play on our servers.

If you are encountering any issues, please see Tech Support so a support represenative can assist you.
